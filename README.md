

# Installation

The project is supported by Python 3, preferred to directly install the Anaconda distribution. Below are a few libraries and packages to install:

    Numpy 1.15.2
    Pandas 0.23.4
    IPython 6.5.0
    Matplotlib 3.0.0
    scikit-learn 0.20.0
    sqlalchemy 1.2.12
    nltk 3.3.0
